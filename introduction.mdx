## Overview

Samosa GG provides a centralized platform where you organize and manage all your project documentation. You create custom spaces, collaborate with teams, and keep everything up to date with version control and real-time editing.

<Callout kind="success">
Samosa GG streamlines your documentation workflow, saving you hours on maintenance.
</Callout>

Samosa GG offers powerful tools that make documentation effortless. You benefit from intuitive interfaces, seamless integrations, and robust search capabilities.

## Get Started in Minutes

You set up Samosa GG quickly with these steps. Follow them to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit samosa.gg and create your account using email or GitHub.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" and name it for your project, like "API Docs".
  </Step>
  <Step title="Add Content" icon="edit-3">
    Start writing MDX pages or import from Markdown files.
  </Step>
  <Step title="Publish" icon="globe">
    Make your space public or share with specific team members.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    Install the CLI for local development:
    ```bash
    npm install -g @samosa-gg/cli
    samosa init my-space
    ```
  </Tab>
  <Tab title="GitHub" icon="github">
    Connect your repo for automatic sync:
    ```bash
    git remote add samosa https://git.samosa.gg/your-org/docs.git
    git push samosa main
    ```
  </Tab>
</Tabs>

## Brand Guidelines

Follow these guidelines to maintain consistency across your documentation.

<ExpandableGroup>
  <Expandable title="Primary Color" default-open="true">
    Use the brand color `{ #3B82F6 }` for accents and links. It represents trust and professionalism.

    Example CSS:
    ```css
    :root {
      --brand-color: #3B82F6;
    }
    a {
      color: var(--brand-color);
    }
    ```
  </Expandable>
  <Expandable title="Typography">
    Stick to sans-serif fonts like Inter or system defaults for readability.
  </Expandable>
  <Expandable title="Icons">
    Use Lucide icons matching the `{ #3B82F6 }` theme for visual consistency.
  </Expandable>
</ExpandableGroup>

<Callout kind="tip">
Customize your space theme with the brand color `{ #3B82F6 }` in settings for a unified look.
</Callout>

<CodeGroup tabs="JavaScript,Python">

```javascript
// Embed Samosa GG docs in your app
```

```python
# Python SDK example
from samosa import Client

client = Client(api_key="YOUR_API_KEY")
space = client.spaces.get("your-space-id")
print(space.render(theme_color="#3B82F6"))
```
