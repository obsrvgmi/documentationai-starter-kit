Samosa is a Social trading layer that unifies execution, analytics, and community letting traders act on data and discussions in real time. Powered by Decibel.

At its core, Samosa is a **Telegram based trading bot and terminal** that puts the market right in your pocket. You can trade spot and perps directly inside Telegram, without switching apps. When you need deeper tools, you can easily move to a full web trading terminal.

Samosa offers a range of features designed to improve efficiency and engagement, including copy trading, yield generating opportunities, group leaderboards, and counter trading. These features allow traders to operate with speed, flexibility, and convenience from one unified platform.

Samosa includes a dedicated **Market Intelligence layer** that supports informed trading decisions. This layer provides comprehensive data and insights such as ticker analysis, liquidation maps, wallet analytics, top traders, most-copied traders, market overviews, spot premiums, Decibel statistics, trader insights, live trader feeds, vault details, etc.

## Get Started in Minutes

You set up Samosa GG quickly with these steps. Follow them to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus" title-type="p">
    Visit samosa.gg and create your account using email or GitHub.
  </Step>

  <Step title="Create a Space" icon="plus" title-type="p">
    Click "New Space" and name it for your project, like "API Docs".
  </Step>

  <Step title="Add Content" icon="edit-3" title-type="p">
    Start writing MDX pages or import from Markdown files.
  </Step>

  <Step title="Publish" icon="globe" title-type="p">
    Make your space public or share with specific team members.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    Install the CLI for local development:

    ```bash
    npm install -g @samosa-gg/cli
    samosa init my-space
    ```
  </Tab>

  <Tab title="GitHub" icon="github">
    Connect your repo for automatic sync:

    ```bash
    git remote add samosa https://git.samosa.gg/your-org/docs.git
    git push samosa main
    ```
  </Tab>
</Tabs>
