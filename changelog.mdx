## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for documentation editing.
- Introduced version pinning for stable deployments.

## Improvements

- Enhanced search performance by `>50%`.
- Optimized image uploads with lazy loading.

## Bug Fixes

- Fixed markdown parsing for nested lists.
- Resolved authentication timeout on long sessions.

```javascript
// New collaboration API example
import { SamosaGG } from '@samosagg/client';

const client = new SamosaGG({ apiKey: 'YOUR_API_KEY' });
await client.docs.share('doc-id', { users: ['user1@example.com'] });
```

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Removed legacy v0.x API endpoints. Migrate to `/v1/docs`.

## New Features

- Core documentation engine with MDX support.
- User authentication and project management.

## Initial Setup

Run the following to get started:

```bash
npm create samosa-gg@latest my-project
cd my-project
npm run dev
```

</Update>
